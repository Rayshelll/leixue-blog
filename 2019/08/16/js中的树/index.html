js中的树

发表于 2019-08-16 | 分类于 前端学习

js树

树是一种非顺序数据结构（节点，根节点，内部节点，外部节点，子树，深度），一般使用递归去处理树。

节点包括（key left right）

二叉搜索树是二叉树的一种，左节点key值<父节点key值，右节点key值>=父节点key值（左<中<=右）

五种遍历的主要思想：
前序遍历：根左右;
中序遍历：左根右;
后序遍历：左右根;
广度优先遍历：按照层次一层层遍历（使用队列来完成遍历访问）;
深度优先遍历：首先以一个未被访问过的顶点作为起始顶点，沿当前顶点的边走到未访问过的顶点；当没有未访问过的顶点时，则回到上一个顶点，继续试探别的顶点，直至所有的顶点都被访问过。
